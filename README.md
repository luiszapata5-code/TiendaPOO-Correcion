# TiendaPOO - Sistema de Ventas

Aplicación de escritorio en Java Swing para gestionar 
clientes, productos y ventas de una tienda de ropa.
Aplica principios SOLID con interfaces y separación por capas.

## Cómo ejecutar
Abrir con NetBeans y ejecutar VentanaPrincipal.java

## Módulos
- Clientes: registrar, editar, eliminar, listar
- Productos: registrar, editar, eliminar, listar
- Ventas: carrito multi-producto, historial de ventas
